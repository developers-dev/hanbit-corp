## Design Context

### Users
대기업/엔터프라이즈 의사결정자 (CTO, IT 디렉터, PM). IT 외주 개발 파트너를 찾는 맥락에서 사이트를 방문. 기술적 신뢰감과 전문성을 빠르게 판단하려 함.

### Brand Personality
정밀 · 신뢰 · 전문. "한 비트도 놓치지 않는다"는 엔지니어링 정밀함이 핵심. 과시적이지 않고 실력으로 말하는 톤.

### Aesthetic Direction
따뜻한 프로페셔널. 부드러운 색감과 둥근 형태, 친근하면서도 전문적. 과하지 않은 모션. 에디토리얼 느낌의 여백과 타이포그래피. 워밍톤 뉴트럴에 딥 네이비/인디고 브랜드 컬러, 따뜻한 코퍼/앰버 악센트.

### Anti-references
- AI 템플릿 느낌: glassmorphism, blue-purple 그라디언트, gradient text, 마우스 따라다니는 blob
- 과도한 애니메이션: 스크롤마다 튀어나오는 요소, 불필요한 모션

### Design Principles
1. **절제된 확신** — 장식이 아니라 구조와 여백으로 전문성을 전달
2. **따뜻한 뉴트럴** — 차가운 slate/blue 대신 warm stone 계열, 인간적 온기
3. **의도적 비대칭** — 센터 정렬 반복 대신 좌정렬과 비대칭 그리드로 디자인된 느낌
4. **의미 있는 모션만** — 페이지 로드 시 한 번의 스태거, 상태 변화 시 트랜지션만
5. **타이포그래피가 주인공** — 아이콘/카드 대신 폰트 크기·굵기·간격으로 위계 설정
